# Información sobre Componentes Blazor.
### Gráfico Interactivo (ChartComponent)
El componente **ChartComponent** proporciona una experiencia de visualización de datos avanzada en tu aplicación Blazor. Permite la presentación interactiva de datos estadísticos mediante gráficos personalizables. Los usuarios pueden explorar y analizar datos fácilmente gracias a las siguientes características:

- **Tipos de Gráficos:** Soporta diversos tipos de gráficos, incluyendo línea, barras, pastel, entre otros.
- **Anotaciones:** Permite agregar anotaciones relevantes en puntos específicos del gráfico.
- **Datos Dinámicos:** Los datos se pueden cargar dinámicamente, adaptándose a las necesidades de visualización.

**USO:**
```
<!-- Ejemplo de uso en Index.razor -->

@page "/"
@using TuProyecto.Shared

<h1>¡Bienvenido a mi aplicación Blazor!</h1>

<ChartComponent />
```

### Carrusel de Imágenes Dinámico (CarouselComponent)
El componente **CarouselComponent** brinda una forma sencilla de mostrar y explorar una colección dinámica de imágenes en tu aplicación Blazor. Ofrece una experiencia visual atractiva y funcionalidades como desplazamiento automático y navegación entre imágenes. Sus características principales incluyen:

- **Imágenes Dinámicas:** Permite la carga dinámica de imágenes, adaptándose a las necesidades del usuario.
- **Desplazamiento Automático:** Ofrece la opción de desplazamiento automático para una experiencia de visualización continua.
- **Navegación Intuitiva:** Permite a los usuarios avanzar o retroceder entre imágenes con facilidad.

**USO:**
```
<!-- Ejemplo de uso en Index.razor -->

@page "/"
@using TuProyecto.Shared

<h1>¡Bienvenido a mi aplicación Blazor!</h1>

<CarouselComponent />
```
</br>

**Nota:** Asegúrate de personalizar y ajustar estos componentes según las necesidades específicas de tu proyecto. La información proporcionada aquí sirve como introducción para que los desarrolladores comprendan las características y puedan integrar fácilmente estos componentes en sus aplicaciones Blazor.
